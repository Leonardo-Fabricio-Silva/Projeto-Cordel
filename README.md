# 📜 Cordel Moderno

Projeto desenvolvido como prática de HTML e CSS, baseado no desafio 12 do canal Curso em Vídeo.

## 🚀 Objetivo

Aplicar conceitos fundamentais de desenvolvimento web, com foco em estruturação, estilização e apresentação visual.

## 🛠️ Tecnologias

- HTML5
- CSS3

## 🎯 O que foi praticado

- Estruturação semântica com HTML
- Estilização com CSS
- Uso de variáveis CSS
- Importação de fontes externas (Google Fonts)
- Tipografia personalizada
- Uso de múltiplas classes (classe base + modificadores)
- Efeito de parallax com background fixo
- Responsividade básica
